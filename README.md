# Fórum Hub - Backend

Este é o backend da aplicação Fórum Hub, onde usuários podem se registrar, autenticar e interagir.

## Tecnologias
- Java 17
- Spring Boot
- Spring Security
- JPA com Hibernate
- H2 Database

## Como rodar
1. Clone este repositório
2. Rode o projeto com o comando `mvn spring-boot:run`
